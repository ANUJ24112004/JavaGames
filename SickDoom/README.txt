========================================
       SHAPE RPG ARENA GAME
========================================

A fast-paced 2D arena game where characters are represented by 
colored geometric shapes.

== INSTALLATION ==

Method 1: Run the installer
--------------------------
1. Double-click on "install_game.bat"
2. Follow the on-screen instructions
3. A desktop shortcut will be created

Method 2: Manual installation
---------------------------
1. Copy all files to any folder on your computer
2. Run "run_game.bat" to start the game

== REQUIREMENTS ==

- Windows operating system
- Java 8 or higher installed
- Minimum 2GB RAM recommended

== HOW TO PLAY ==

- The game offers three exciting modes:
  * Campaign Mode: 15 progressive levels with bosses
  * Endless Mode: Wave-based survival with increasing difficulty
  * PvP Mode: 2 players battle each other and enemies

- Controls:
  * Player 1: WASD to move, Space to attack
  * Player 2: Arrow keys to move, Enter to attack
  * P: Pause
  * ESC: Return to main menu

== TROUBLESHOOTING ==

If the game doesn't run:
1. Make sure Java is installed on your computer
2. Try running "java -version" in command prompt to verify
3. You can download Java from https://www.java.com/download/
